# twitter-contest-bot
Will poll for Retweet Contests and retweet them. Inspired by http://www.engadget.com/2015/08/10/twitter-bot-wins-a-thousand-contests/

Prerequisites
------------

  * TwitterAPI
  * Python 2.7

Installation
------------
From the command line:

	pip install TwitterAPI
	
Then run:

	python main.py
